# Bird Sighting Application
This design has been built using React library with React Bootstrap. 
All the components are independent. Redux is used to store the state.
User can add a new bird, update a bird and delete a bird and also add various date and time for a single type of bird.

## Instructions

`git clone `

`npm install`

`npm run start` & Enjoy the new look in your browser.

## Tools used

* React 16.7
* React-Redux 7
* Redux  
* Bootstrap 4
* React Modal for modal
